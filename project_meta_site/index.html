  • Redesign Prototype Site

Our team was formed by Priscila Oliveira, Julia Wu and Jacob Golobokov.

Initially it was discussed with the team what improvements could be implemented, what usability we lacked in comp1950, and references to layouts.

Then, we listed 10 implementations that we believe are necessary for the site:

  1. Rewrite the code in HTML5 - To improve semantic and SEO
  2. Add CSS reset - To make sure the site looks the same across all browsers
  3. Adding a Print CSS - To improve printed content. As well as facilitating the studies of the students, being able to print all content in paper or in digital file.
  4. Making the site responsive - To improve usability across platforms
  5. Change the color scheme - To create better contrast and improve aesthetics
  6. Adding a homepage button - Currently the homepage disappears after user navigates away from it.
  7. Adding a search bar - Easy for users to find the materials they’re looking for (note: this feature is not fully functioning)
  8. Adding a custom error 404 page - To give user a way to redirect back to the homepage
  9. Adding a sidebar - It is a important implementation, because allows user to find lectures and sections easily without clicking or scrolling too many times
  10. Rewriting CSS using SASS - This allows future changes to be made easily and keeps the look unified through each change.
